== Description ==

Tired of using Media Wiki? Sick of fighting with NXTClass to get your own documentation project happening?

NXTClass Wiki Plugin is the answer for you!

The new NXTClass Wiki plugin made by the guys at Instinct (who brought you the famous <a title="nxtclass e-commerce plugin" href="http://www.instinct.co.nz/e-commerce">Wordpress e-Commerce Plugin</a>) have developed a new plugin that adds Wiki functionality to your NXTClass powered website.

== Installation ==

1. Copy the Plugin files to <code>nxt-content/plugins/<code>

2. On NXTClass 2.8-2.9, mark a page as a Wiki to get started. On NXTClass 3.0 and above, start by adding a new Wiki Page.

== Frequently Asked Questions ==

-How do I format text in the front-end editor?
NXTClass Wiki uses the same syntax as MediaWiki. You can learn more here: http://www.mediawiki.org/wiki/Help:Formatting
